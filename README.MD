# Aplication for Scrap USD Price with relation in ARS

  * RedisLabs - cache management
  * MongoDB

## Tools 

  * NodeJs
  * Express
  * Jest - Test
  * RedisLabs
  * moongose
  * cheerio

## Releases

  * Imp - Found prices with scrapper
  * Imp - Save in cache results
  * Serve endpoints since cache db
